# Replit Agent Prompt — Freedom Square USA Landing Page

Build a polished, responsive one-page landing page for **Freedom Square USA** using **React + Vite + Tailwind CSS**.

The landing page is for a membership community called **Freedom Square USA**. The page should persuade visitors to become an **Apprentice Member**. Do not include multiple membership tiers. Apprentice is the only public offer right now but other membership options should be easily added after.

## Project Goal

Create a serious, hopeful, patriotic, conversion-focused landing page for Americans who still believe freedom is worth learning, living, and preserving.

The page should feel:

* hopeful
* steady
* principled
* useful
* patriotic
* warm
* serious without being heavy
* community-oriented

The page should not feel:

* angry
* partisan
* campaign-like
* corporate
* flashy
* apocalyptic
* cluttered
* like a generic membership template

## Tech Stack

Use:

* React
* Vite
* Tailwind CSS
* JavaScript, not TypeScript

Do not add unnecessary libraries. Keep the site lightweight and fast.

## Deployment Assumption

This project will be built in Replit, pushed to GitHub, and deployed on Cloudflare Pages.

The page should work as a static site.
